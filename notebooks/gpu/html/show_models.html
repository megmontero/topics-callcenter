Stacking¶

La ltima arquitectura que presentamos tiene cmo proposito combinar diferentes modelos, pensado inicialmente para combinar modelos binarios, con el objetivo de obtener un nuevo modelo ms potente.

In [6]:
plot_model("stacked_model")
Out[6]:

Vemos que el input alimenta a los diferentes modelos (5 en este caso), que pueden estar construidos tanto con el modelo Model de keras como con el modelo Sequential.


La salida de los diferentes modelos se concatena y, antes de la salida, se aade una capa oculta totalmente conectada con funcin de activacin relu. Normalmente este tipo de modelos no vamos a usarlos en clasificacin binaria por lo que la funcin de activacin de la capa de salida ser sigmoid.


Hay que destacar que los sub-modelos que forman parte de este modelo estan preentrenados y no volvern a entrenarse en el proceso de aprendizaje.
