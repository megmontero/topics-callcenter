En el segundo caso vemos que an poniendo en el conjunto de test un 20%, el nmero de muestras es de unas 15 mil, menor que las 22 mil que tenemos al no balancear. Esto se debe a que al balancear tenemos tantas muestras como la clase que menos tenga (de las que queremos clasificar) multiplicado por el nmero de clases.

Cargar modelo

Para cargar el modelo utilizamos la clase Optimize del mdulo mgmtfm para obtener el mejor entrenamiento de este modelo. Vemos cuales son sus mejores parmetros y el posible nombre para el fichero de pesos que Optuna almacena para su mejor configuracin.

In[7]:
opt_keras = optimize.Optimize(model_name, OPT_BBDD)
best_params = opt_keras.get_best_params()
bt =opt_keras.get_best_trial()
model = models.Models(len(balancing_types)+1,max_seq_len,embedding_steps.embedding_dim,
                            embedding_steps.vocabulary_size,embedding_matrix)
model.get_best_model(model_name, best_params)
prob_fname = "{}_model_{:06d}.h5".format(model_name, bt)
print("Cargar el archivo de pesos del estudio {} y el trial: {}  (probablemente {}) o entrenar.".format(model_name,bt, prob_fname))
print(best_params)
